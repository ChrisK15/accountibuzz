**1. [Rule 3 — Missing dependency] Installed expo-camera, expo-video, expo-blur, react-native-reanimated + react-native-worklets**
- **Found during:** Task 3 (`pnpm test` failed with `Cannot find module 'expo-camera' from 'jest.setup.ts'`)

  - `npx expo install react-native-reanimated` → resolved to `4.2.1`
  - `npx expo install react-native-worklets` (Reanimated 4.x peer; required by `babel-preset-expo` to load Reanimated's babel plugin)
- **Files modified:** `package.json`, `package-lock.json`
- **Verification:** After installs, `npx jest` passes 120 of 120 actual tests across 25 of 26 suites. The 1 failing suite is a pre-existing vitest test in `design_refs/` (not caused by this plan). Pre-edit baseline (Task 3 changes stashed) confirms identical 1-suite failure pattern, ruling out regressions.
